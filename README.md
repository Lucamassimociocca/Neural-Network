# Neural-Network (TensorFlow/Keras — Regression & Classification)
This repository implements feedforward neural networks using TensorFlow/Keras for both regression and classification tasks. The architecture follows a sequential, fully connected design: an input layer receives a standardized feature vector and passes it through one or more hidden layers of dense neurons with ReLU activation functions, which introduce non-linearity while mitigating the vanishing gradient problem. For regression tasks, the output layer uses a linear activation function to produce continuous predictions; for classification tasks, a sigmoid activation produces class probabilities. Hyperparameter optimization is handled automatically via Keras Tuner's Hyperband algorithm, which searches over the number of hidden layers (1–3), neurons per layer (32–256), dropout rate (0.0–0.5), and learning rate (10⁻⁴–10⁻²). Regression models use Huber loss for robustness to outliers, with an optional QuantileTransformer on skewed targets; classification models use binary cross-entropy optimized for AUC. All models use dropout regularization and early stopping to prevent overfitting, and are evaluated on a held-out test set (20%) using MAE, R², or ROC AUC depending on the task. Post-training, partial dependence plots and permutation feature importance are computed to interpret which variables drive predictions. For classification, SMOTE is applied to the training set to address class imbalance, and the classification threshold can be adjusted to prioritize recall over precision as needed. The feature set and target variable are defined at the top of the script and can be freely substituted for any tabular dataset.
